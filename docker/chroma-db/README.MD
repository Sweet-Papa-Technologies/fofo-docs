# Chroma DB Docker Container:


Use this to quickly spin up a local Chroma DB server via Docker. Of course ensure you have Docker installed on your machine.

## Instructions:
Edit the .env file:
```
API_PASS=YOPASSWO
API_USER=YUSERNAME
```

Run the following command:
```
sh run.sh
```

## Access:
```
http://localhost:8000
```

Docker Name: fofo-chroma-db
